# Smartlab-Repository

With this repository comes the ability to set up a homeautomation-system, based on openHAB and Z-Wave. It also provides a ROS-bridge and several addons 


Table of content:
| core    | addons         |
|---------|----------------|
| openHAB | ROS-bridge     |
| HABmin  | gcal-parser    |
|         | e-mail client  |
|         | emoncms client |
